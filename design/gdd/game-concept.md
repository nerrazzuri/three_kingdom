# 游戏概念

## 高层概念

《三国演义：兵法沙盒》是一款离线单机三国沙盒战略 RPG。玩家不是悬浮于世界之外的全知君主，而是**以「城池太守」为固定开局身份**，生活在势力、城市与关系网络中的人物。天下大势严格沿《三国演义》时间线推进（**大势随历史**），而玩家个人的城池归属、官职升降、势力命运由自己的抉择与实战塑造（**个人靠抉择**）。玩家通过经营人生与局势取得资源、信任和情报，再在战争中利用真实形成的条件取胜。

## 核心幻想

玩家希望感到：“这不是系统送给我的火攻技能，而是我先探明粮营、拖到干燥季节、诱敌进入下风口、稳住己军纪律，才让火攻成为可能。”同样的计策在不同地形、天气、人物和政治环境中可能成立、失败或产生意外后果。

## 玩家定位与开局（锁定）

- **固定开局身份：城池太守。** 玩家空降为某正规势力的一城太守（魏 / 蜀 / 吴 / 袁绍 / 刘表 / 吕布等演义势力，随机或可选）。开局城池匹配历史禀赋——专属文武团队、固定兵力、城池等级、资源产出、城防各不相同，每城开局独一无二。
- **开局即守城战（强制剧情）。** 进入游戏即触发历史敌军围城，无新手碾压、无跳过——玩家须亲自排兵布阵、手动指挥守城。
  - **胜**：解锁全部城池权限、正式开启势力生涯、获君主初始信任与功绩。
  - **败**：城池沦陷、官职罢免、沦为在野武将，保留少量核心部曲，可重新投效势力或流浪发育（败局**必须**生成合法可继续状态，不卡死）。
- **双线自由抉择（互不冲突，可随局势切换）：**
  - **忠臣晋升线**：依附所属君主，完成朝廷政令任务、积累功绩名望，从太守逐级晋升，最终官至大都督乃至继承基业。
  - **自立反叛线**：不靠官职晋升，凭实战硬实力与文武僚属好感，伺机脱离主君自立为王；最终结局由关系网决定（被拥立 / 部分跟随 / 众叛亲离）。

## 游戏三层结构（锁定）

游戏由三层叠成，缺一层就不成立——前两层是这套设计长期缺失、最易跑偏之处，特此锁定：

1. **历史世界模型**：《演义》时间线提供“谁占哪城、谁打谁、下一步打什么”的世界骨架。历史事件按**前置条件**触发——条件还在则历史继续；玩家强大到改变了某事件的前置条件，则该事件分叉（如提前灭孙权则赤壁不成立）。玩家够不着的事，前置条件恒成立（详见 GDD_015）。
2. **个人生涯**：太守 →（忠臣）晋升梯队 /（自立）割据争霸。名望与功绩可兑换官职、兵权、地盘、自立资格——**这是可玩性、连接性、持续性的来源**（详见 GDD_014）。
3. **兵法战斗**：在“战区”中靠**布局条件**（地形 / 天气 / 时间 / 补给 / 人心 / 情报）创造胜机；伏击、断粮、火攻、夜袭等是条件链**涌现**的结果，**不是技能按钮**（详见 GDD_010）。

## 核心循环

1. 生活与观察：处理身份、职责、关系、城市和天下事件。
2. 判断与布局：侦察局势，争取人物、物资、权限和外交空间。
3. 行动与承诺：发布命令、投入时间和资源，暴露风险。
4. 战争与应变：根据不完全信息执行部署，观察反馈并调整。
5. 结果与延续：战果改变名声、关系、城市、派系与战略态势，失败进入新的可玩局面。

## 独特性（护城河）

- 区别于传统三国大战略：重点不是以势力面板进行全知管理，而是从人物人生出发塑造战争条件。
- 区别于 Total War-like：不以即时大规模军阵操控和战斗表现为核心产能中心，而以跨系统因果和兵法条件创造为核心。
- 区别于技能制战术 RPG：计策不是冷却技能；它们是对局势的判断、准备过程与风险承诺。
- **核心护城河**：兵法沙盒战斗（条件涌现）+ 不完全情报 + 太守人生三者合一。其中“人心杠杆”（离间 / 策反 / 攻心）把战斗接入关系与情报系统，是 Total War / 三国志都没有的独门差异。

## 与《三国志》系列的关系（参考，非复制）

本作**参考**《三国志》系列已验证的系统**存在性**（城池经营、武将关系、官职晋升），但**不复制其全套深度**，更不以势力面板做全知管理。

**砍 scope 的尺子（强制，防“参考”悄悄变“抄袭”）**：每新增一个经营 / 关系 / 官职功能，必须能回答“**它喂给战斗或生涯抉择什么**”。喂不到的即“抄”，砍掉；喂得到的才是“参考”，保留。城池系统遵循 `game-pillars` 最低承诺——只需**改变补给 / 征募 / 防御 / 情报质量**，**不**做成经营模拟器。

## 设计边界

- 强调战争特色，但非战斗系统必须占据有意义的决策空间。
- 允许军师建议，但建议必须保留不确定性、偏见和玩家决策权。
- 允许戏剧化表达，但规则结果必须能被解释和复盘。
- 允许玩家失败，但必须提供撤退、求和、失职、流亡、投效或东山再起等延续路径。

## 当前设计结论

本项目是“**兵法沙盒三国人生大战略**”：以城池太守开局，大势随《演义》、个人靠抉择，靠在战区中创造战争条件以弱胜强，经忠臣晋升或自立争霸延续人生。**不是**“三国志复制品”，**不是**全知君主面板，也**不是**把三国题材覆盖在技能战斗上的产品。
