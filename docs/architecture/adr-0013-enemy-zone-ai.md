# ADR-0013 — 敌方区域AI效用模型（Enemy Zone Battle AI）

> **Status**: Accepted（2026-07-04；lean 内联裁定——特化 ADR-0006 于区域战斗层，无新违反禁则，story 可引用。首次落地 GDD_016 敌方AI。）
> **相关**: GDD_021 区域战斗 · GDD_016 敌方AI · epic-031 · ADR-0006（确定性效用敌方AI，基座）· ADR-0012（区域战斗引擎）· ADR-0004（确定性）

## Context（背景）

用户裁定：战斗是核心玩法、须最完善，敌方AI区域博弈**必做且一步到位做完善**。GDD_016（敌方AI）此前 Reviewed 但从无实现——本 ADR 让它在区域战斗层**首次落地**。核心问题：如何让敌方在多区域战斗中做出**有深度、不可预测却可复现、不作弊（反全知）**的区域决策，且满足确定性（ADR-0004）。

## Decision（决策）

**D1 特化 ADR-0006 于区域动作空间，不新造随机基座。**
沿用 ADR-0006 的"效用评分 + 种子化 softmax + 反全知锁 + LLM 隔离"。本 ADR 只定义**区域战斗的动作空间、因子、种子与记忆**。随机只走注入式确定性流（ADR-0006 同款），非掷骰。

**D2 反全知：敌AI只见 `AiWorldView` 区域投影，不接受真值。**
敌方对玩家各区的兵力/部署/隐蔽只有**其自身情报投影**（可错、可过时）。玩家隐蔽区（伏兵）未被敌侦破 → 敌视野中不存在（不会"预判"未知伏兵）。**结构上拒绝真值注入**（延续 ADR-0006 反全知锁 + GDD_021 R8/R9）。

**D3 区域动作空间（MVP，最完善向迭代）。**
每回合敌方对每区/全局评估动作：`增援区X` / `反扑区Y` / `佯动` / `设伏于某区` / `收缩/撤离某区` / `断我粮道` / `按兵不动`。动作合法性同玩家（相邻+在途，ADR-0012 D3）——**敌方受同规则约束，不作弊**。

**D4 效用评分（数据驱动因子）。**
`U(action) = Σ w_i · factor_i`，因子含：区域战略价值、己方该区相对兵力、受威胁度、情报置信度、突破/被突破预期、粮道安危。权重数据驱动（可调、可平衡）。定点整数（ADR-0004）。

**D5 种子化 softmax 选择 + 温度。**
`P = softmax(U / τ)`；`seed = Hash(worldTick, battleRound, factionId, actionCandidateId)`；注入式确定性流取样 → **不可预测却可复现**（存读档/回放一致）。温度 τ 可调（低=更优、高=更诡）。

**D6 渐进记忆。**
敌方记住玩家近 K 回合区域动作（如"玩家连续增援正面"）纳入因子（预期玩家意图），形成多回合博弈感。记忆入 `ZoneBattleState`、随存档、确定性。

**D7 LLM 隔离。**
LLM 仅可生成敌将台词/战报等**装饰性文本**，**绝不**参与效用/选择/胜负结算（延续 ADR-0006）。

**D8 分期：MVP"能博弈"→ 迭代"最完善"，但架构一次到位。**
动作空间/因子/记忆的**架构**一次设计完（可扩展），MVP 先实装核心动作 + 基础因子（已强于三国志固定套路），后续按平衡迭代加高级诈术（诱敌深入/多回合佯败/兵种协同），无需重构。

## Consequences（后果）

**正面**
- 敌方成为真正的区域对手（反制、设伏、断粮、记忆玩家）→ "比三国志更能排兵布阵"的深度来源。
- 反全知 + 同规则约束 → 敌不作弊，胜负靠布势博弈而非数值碾压。
- 种子化 → 不可预测却可复现，守 ADR-0004/0006，可存档续战。

**负面 / 代价**
- 效用权重/温度平衡面大 → 需大量对局测试与平衡用例（战斗须"最完善"，投入必要）。
- 记忆 + 多因子 → AI 决策测试须覆盖确定性/反全知/记忆生效多组断言。
- "最完善"是迭代目标：MVP 后仍需数轮 AI 深度迭代（分期，D8）。

## Alternatives Considered（备选）

- **A 固定脚本敌方（如竖切）**：可预测、无博弈 → 否决（战斗是核心，须最完善）。
- **B 全知最优 AI**：强但作弊、可预测、无戏剧性 → 违反反全知 → 否决。
- **C 本决策：反全知 + 种子softmax效用 + 记忆 + 分期迭代** → 采纳。

## Compliance / Guardrails

- 随机只走注入式确定性流（ADR-0006）；种子含 worldTick/battleRound/faction/action。
- 反全知锁：结构上拒绝真值注入敌视野；未侦破隐蔽区不可见（断言）。
- 敌方受与玩家同的命令合法性约束（相邻+在途，不作弊）。
- LLM 仅装饰，不参与结算（断言：结算路径无 LLM 依赖）。
- 定点整数、无 float 权威路径（ADR-0004）。
