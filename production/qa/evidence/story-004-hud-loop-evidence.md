## 人工走查（待用户在 Unity Editor 完成并签核）

Setup：MainMenu「新游戏」→ 进入 HUD。按序操作并勾选（1080p 与 1440p 各查一遍可读性）：

- [ ] **顶栏相位横幅**：显示「相位：治理｜可做：推进时段、征用军粮、修工事、安抚、侦察、召开军议、设伏备战」
- [ ] **治理**：己方账本分列显示 粮草/民心/治安/工事（无单一「实力」总分）；征用/修工事/安抚三按钮各旁有因果方向说明（如「修工事：↑ 守城工事强度」）
- [ ] **点「征用军粮」**：账本粮草↓、民心↓；点「修工事」工事↑；点「安抚」民心↑
- [ ] **点「征用军粮」多次至超量**：显示稳定错误码文案（如「粮草可分配量不足」），账本不因失败而变
- [ ] **备战**：点「设伏（加入草稿）」→ 草稿区出现可「移除」按钮；「提交计划」后草稿转为「朱批已承诺」（不可移除），状态显示「已提交…不可反悔」
- [ ] **相位转「备战」→ 提交后「开战」可点**
- [ ] **点「开战」**：进入「战中」相位；战况区显示兵法条件清单（✓/✗ + 还差 N 条），非按钮；「结算战果」可点
- [ ] **点「结算战果」**：进入「战后」相位；下方战果复盘出现（story-002 折叠一句话，可展开）；续局选项可选
- [ ] **全程**：每一步顶栏「可做」都指明下一步能做什么，无需查文档

**Pass 条件**：全程无单一实力数字；无「游戏结束/胜率」；任一相位都看得到「现在能做什么」；备战提交后视觉变化明显。
